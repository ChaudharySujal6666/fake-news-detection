# 📰 AI Fake News Detection System

## Overview
This project is a Machine Learning-based Fake News Detection System that classifies news articles as Fake or Real using Natural Language Processing (NLP).

## Features
- Fake and Real News Classification
- Text Preprocessing
- Stopword Removal
- Stemming
- TF-IDF Vectorization
- Logistic Regression Model
- Custom News Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Joblib
- Google Colab

## Dataset
ISOT Fake News Dataset

## Model
- Logistic Regression
- TF-IDF Vectorizer
